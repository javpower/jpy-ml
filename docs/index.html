Why jpy-ml
传统方案 vs jpy-ml
不是又一个 REST 包装器。jpy-ml 通过 JNI 将完整 Python ML 生态嵌入 JVM，零网络延迟，类型安全。

😤 传统 Java ML

  • ❌ REST 调用 Python 服务器 — 网络延迟 + 序列化开销
  • ❌ 手动安装 Python + pip + PyTorch
  • ❌ 解析无类型的 JSON — Map<String, Object> 满天飞
  • ❌ 部署 2 个服务（Java 应用 + Python API）
  • ❌ 只能用 ONNX Runtime（CPU only）
  • ❌ 只有推理，没有训练、验证、导出

⚡ jpy-ml

  • ✅ ML 通过 JNI 进程内运行 — 零网络延迟
  • ✅ 自动下载 Python、依赖和模型权重
  • ✅ 强类型结果：DetectionResult、PoseResult...
  • ✅ 单 JVM 进程 — 运维简单、成本更低
  • ✅ 完整 PyTorch — CPU / MPS / CUDA / 多卡
  • ✅ 推理 + 训练 + 验证 + 导出 — 完整生命周期

    DetectionResult result = model.predict("photo.jpg");

    for (ClassPrediction box : result.getBoxes()) {
        System.out.printf("%s (%.1f%%) at %s%n",
            box.className(), box.confidence() * 100, box.box());

    String json = result.toJson();

}
// 实例分割 — 自动检测 + 掩膜
try (Model model = Model.preset("yolov8n-seg")) {
    SegmentationResult result = model.predict("photo.jpg");

    // 获取所有掩膜多边形
    for (Mask mask : result.getMasks()) {
        System.out.println("Mask points: " + mask.getPointCount());
    }

    // 掩膜可视化
    ImageVisualizer viz = new ImageVisualizer().maskAlpha(0.5f);
    BufferedImage annotated = viz.visualize(image, result);
}
// 姿态估计 — COCO 17 关键点
try (Model model = Model.preset("yolov8n-pose")) {
    PoseResult result = model.predict("person.jpg");

    System.out.println("Persons: " + result.personCount());

    for (KeypointCollection kc : result.getKeypoints()) {
        Keypoint nose = kc.getNose();
        System.out.printf("Nose: (%.1f, %.1f) conf=%.2f%n",
            nose.x(), nose.y(), nose.confidence());
    }
}
// SAM 2 — 点击分割 + 视频追踪
try (SAM2Model sam = new SAM2Model("sam2.1_t.pt")) {
    // 点 Prompt 分割
    SAM2Result result = sam.predict("photo.jpg",
        Prompt.point(320, 240));
    System.out.println("Best score: " + result.bestScore());

    // 视频追踪
    try (SAM2VideoTracker tracker = sam.trackVideo("video.mp4",
            Prompt.box(100, 100, 400, 400))) {
        SAM2VideoResult video = tracker.propagate();
        System.out.println("Tracked: " + video.trackedFrameCount() + " frames");
    }
}
// 训练自定义模型 — 含 Epoch 回调
try (Model model = new Model("yolov8n.pt")) {
    TrainingConfig config = new TrainingConfig()
        .dataConfig("coco128.yaml")
        .epochs(50)
        .imageSize(640)
        .device(Device.cuda(0));

    TrainingResult result = model.train(config, (epoch, log) -> {
        System.out.println("Epoch " + epoch + ": " + log);
    });

    // 验证
    ValidationResult val = model.validate("coco128.yaml");
    System.out.printf("mAP50=%.3f P=%.3f R=%.3f%n",
        val.getMAP50(), val.getPrecision(), val.getRecall());

    // 导出 ONNX
    ExportResult exported = model.export(ExportFormat.ONNX);
}

Quick Start
3 分钟上手
添加依赖 → 写代码 → 运行。就这样。
1

添加 Maven 依赖

<dependency>
    <groupId>io.github.javpower</groupId>
    <artifactId>jpy-ml</artifactId>
    <version>1.0.0</version>
</dependency>
2

初始化 & 推理

// 初始化 Python 运行时（自动下载 Python + 依赖）
PythonRuntime.init();

// 加载模型并推理（自动下载模型权重）
try (Model model = Model.preset("yolov8n")) {
    DetectionResult result = model.predict("photo.jpg");
    result.getBoxes().forEach(System.out::println);
}
3

运行

mvn clean compile exec:java

# 或使用本地 venv（开发模式）
PythonRuntime.init(pythonPath, jepLibraryPath);
